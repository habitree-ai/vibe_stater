> **용도**: Habitree 서비스 내 초보자용 AI 활용 가이드/강의 콘텐츠 기획
> **대상**: (a) 완전 입문 일반인 · (b) 직장인·자기계발 · (c) 시니어·디지털취약층
> **원칙**: ① 비용 0원(무료 티어/무료 도구) ② 몇 분 안에 따라 가능 ③ '아하 모먼트'가 큰 것 위주
> **기준 시점**: 2026년 7월 (⚠️ 무료 티어 한도·모델명은 자주 바뀌므로 콘텐츠에 "○년 ○월 기준" 명시 필수)
> **근거**: 딥리서치(웹 26개 출처 → 113개 주장 추출 → 25개 교차검증 → 16개 확정). 문서 맨 아래 [출처·검증 상태] 참조.

## 1. 한 줄 결론 (기획 방향)

> **초보자가 무료 AI에서 가장 크게 "우와" 하는 지점은 '정답 생성기'가 아니라 ①내 자료를 넣고 ②나 대신 정리·번역·연습시켜 주고 ③이미지에 정확한 한글 글자를 넣어 주는 순간이다.** Habitree의 따뜻하고 실용적인 톤으로, "코딩 몰라도" 처럼 "AI 몰라도 오늘 당장" 체감시키는 게 핵심.

### 초보자 '아하 모먼트' Top 3 (검증된 근거 기반)
1. **공부 모드 = 개인 과외 선생님** — ChatGPT 공부 모드가 정답을 바로 주는 대신 질문·힌트로 유도. `[검증됨]`
2. **내 강의노트/PDF → 자동 퀴즈·플래시카드** — Gemini 무료가 업로드 자료로 시험대비 문제 생성. `[검증됨]`
3. **말로 설명 → 한글 글자가 또박또박 박힌 포스터/썸네일** — Gemini(Nano Banana)가 이미지 안에 정확한 텍스트 렌더링. `[검증됨]`

## 2. 먼저 정리하는 "0원으로 시작하기" (모든 강의의 1강 소재)

세 가지 대표 챗봇 모두 **신용카드 없이** 영구 무료로 시작 가능. `[검증됨]`

| 도구 | 접속 | 무료로 되는 것 | 무료 한도(대략, 변동성 큼) |
|------|------|----------------|----------------------------|
| **ChatGPT** | chatgpt.com | 대화, 파일·이미지 업로드, 웹검색, **공부 모드**, 이미지 생성(소량), 커스텀 GPT '사용' | 기본 모델 제공 후 한도 초과 시 경량 모델로 다운그레이드. 이미지 생성 약 2~3장/24h `[검증됨: medium]` |
| **Claude** | claude.ai | **카드·체험기간 없이 바로 대화**, 웹검색, 파일 업로드, Projects, Artifacts | 약 5시간당 15~40 메시지 수준. 최상위 모델은 유료 전용 `[검증됨]` |
| **Gemini** | gemini.google.com | 대화, **업로드 자료로 퀴즈·플래시카드·학습가이드**, 이미지 생성/편집, 음성(Live), NotebookLM 접근 | 컴퓨트 기반 한도(수 시간마다 리셋, 주간 상한) `[검증됨]` |

> **강의 톤 팁**: "셋 다 공짜예요. 오늘은 딱 하나만 골라 시작해요. 뭘로 시작할지 고민되면 → 글·정리·번역은 아무거나, **공부/시험 자료는 Gemini(내 파일 넣기)**, **이미지·포스터는 Gemini 이미지**." 로 선택 부담을 없애준다.

⚠️ **1강에 반드시 넣을 안전 문구 2개** (아래 8장 상세):
1. "채팅창에 **주민번호·계좌·비밀번호·회사 기밀은 넣지 마세요.** 무료 대화는 기본적으로 AI 학습에 쓰일 수 있어요(설정에서 끌 수 있어요)." `[검증됨]`
2. "AI는 **가끔 자신 있게 틀려요(환각).** 숫자·의료·법률·날짜는 꼭 한 번 더 확인하세요." `[검증됨]`

## 3. 영역별 활용법 (콘텐츠 본문 재료)

각 항목: **무료 도구 → 프롬프트 예시(복붙용) → 초보자 체감 효과 → 주의점**

### 3-1. 일상생활 🏠

#### ① 긴 글/유튜브/약관 3줄 요약
- **도구**: ChatGPT·Claude·Gemini 무료 (텍스트 붙여넣기/링크·파일) `[검증됨: 파일업로드 무료 가능]`
- **[예시] 프롬프트**:
  ```
  아래 글을 초등학생도 이해할 만큼 쉬운 말로 3줄 요약해줘.
  그리고 '내가 꼭 알아야 할 것' 1가지를 콕 집어줘.
  ---
  (여기에 글/약관/기사 붙여넣기)
  ```
- **체감 효과**: 읽기 귀찮던 걸 30초에 파악 → 가장 흔한 첫 성공 경험.
- ⚠️ 요약이 원문을 왜곡할 수 있으니 중요한 계약/의료는 원문 확인.

#### ② 번역 + "자연스럽게 다듬기"
- **도구**: 무료 챗봇 아무거나
- **[예시] 프롬프트**:
  ```
  이 문장을 영어로 번역하고, 원어민이 실제로 쓰는 자연스러운 표현으로 바꿔줘.
  격식(비즈니스) 버전과 친근한 버전 2가지로 줘.
  "다음 주 미팅 일정을 조정하고 싶습니다."
  ```
- **체감 효과**: 번역기보다 "상황에 맞는" 문장이 나옴.

#### ③ 냉장고 재료로 오늘 저녁 / 여행 일정 짜기
- **[예시] 프롬프트**:
  ```
  냉장고에 계란, 대파, 김치, 두부가 있어. 20분 안에 만들 수 있는
  저녁 메뉴 3개를 재료·순서와 함께 알려줘. 맵기는 약하게.
  ```
- **체감 효과**: "나한테 맞춰서" 답이 나온다는 개인화 경험.
- ⚠️ 건강·복약·알레르기 정보는 AI 답을 맹신하지 말고 전문가/공식 정보 확인(환각).

#### ④ 공감 메시지/축의금 문구/사과 문자 대신 써주기
- **[예시] 프롬프트**:
  ```
  친한 친구 부모님상에 보낼 위로 문자를 3가지 톤으로 써줘.
  너무 격식 차리지 않고 진심이 느껴지게, 각 2~3문장.
  ```
- **체감 효과**: "말이 안 떠오를 때" 막막함 해소 — 감정 노동을 덜어줌.

---

### 3-2. 업무/생산성 💼

#### ① 이메일 초안 → 톤 조절
- **[예시] 프롬프트**:
  ```
  아래 상황으로 거래처에 보낼 정중한 이메일을 써줘.
  - 상황: 납기 3일 지연, 사과 + 새 일정(다음 주 화) 제안
  - 톤: 정중하지만 너무 굽신대지 않게
  - 제목 3개 후보도 같이
  ```
- **체감 효과**: 백지 공포 해소. 초안 10분 → 1분.

#### ② 회의록/녹취 정리 → 할 일 추출
- **[예시] 프롬프트**:
  ```
  아래 회의 메모를 정리해줘.
  1) 핵심 결정사항  2) 담당자별 할 일(마감 포함)  3) 다음 회의 안건
  표로 깔끔하게.
  ---
  (메모 붙여넣기)
  ```
- **체감 효과**: 정리 노동이 사라짐 → 직장인 만족도 가장 높은 항목.
- ⚠️ 회사 내부 회의 내용은 민감정보. 익명화하거나 데이터 학습 옵트아웃 후 사용.

#### ③ 엑셀 함수/수식 물어보기
- **[예시] 프롬프트**:
  ```
  엑셀에서 A열 날짜가 이번 달이면서 B열 금액이 10만원 넘는 행의
  합계를 구하는 수식을 알려줘. 초보자용으로 각 부분이 뭔지 설명도 같이.
  ```
- **체감 효과**: 검색으로 30분 헤매던 걸 즉답 + 설명.
- ⚠️ 결과 수식은 실제 데이터로 한 번 테스트(환각 가능).

#### ④ 자료조사 초안 (그다음 사실 확인)
- **도구**: Gemini(월 5회 Deep Research 무료 포함) 또는 웹검색 켠 챗봇 `[검증됨]`
- **[예시] 프롬프트**:
  ```
  '2030 세대 대상 무료 뉴스레터 마케팅 사례'를 조사해서
  표(사례명 / 방식 / 특징 / 출처링크)로 정리해줘. 출처 링크 꼭 포함.
  ```
- ⚠️ **출처 링크를 반드시 클릭해 실재 확인** — AI가 가짜 링크/통계를 만들 수 있음.

> **PPT/보고서 자동화**: ChatGPT로 목차·본문·표를 만들어 워드/노션에 서식 유지하며 붙여넣는 워크플로가 초보 직장인에게 인기(오빠두엑셀 등 국내 실전 가이드 존재). 단, 구체적 무료 도구별 한도는 이번 검증에서 확정 못 함 → [9장 추가 확인] 참조.

### 3-3. 학습/자기계발 📚

#### ① ChatGPT 공부 모드 = 무료 개인 과외 ⭐ (최고의 아하 모먼트)
- **도구**: ChatGPT 무료 포함(2025-07-29부터). 정답을 바로 안 주고 소크라테스식 질문·힌트로 유도. `[검증됨]`
- **[예시] 프롬프트**:
  ```
  공부 모드로 도와줘. 나는 '금리와 환율의 관계'를 이해하고 싶은 완전 초보야.
  한 번에 다 설명하지 말고, 내 수준을 질문으로 확인하면서
  단계별로 이끌어줘. 중간중간 이해했는지 퀴즈도 내줘.
  ```
- **체감 효과**: "떠먹여 주는 답"이 아니라 "이해시키는 선생님" — 학습 지속에 결정적.

#### ② 내 강의노트/교재 PDF → 자동 퀴즈·플래시카드 ⭐
- **도구**: Gemini 무료 (파일 업로드 → 퀴즈/플래시카드/학습가이드 생성) `[검증됨]`
- **[예시] 프롬프트**:
  ```
  첨부한 강의노트로 시험 대비를 도와줘.
  1) 핵심 개념 10개 플래시카드(앞:용어 / 뒤:쉬운 설명)
  2) 객관식 5문제 + 정답·해설
  3) 내가 자주 틀릴 만한 함정 포인트
  ```
- **체감 효과**: 시험공부 준비 시간 대폭 단축, 학생·수험생·자격증 준비층에 강력.
- ⚠️ 생성된 문제/정답도 오류 가능 → 교재로 교차 확인.

#### ③ 무료 영어(외국어) 회화 파트너
- **도구**: ChatGPT/Gemini 음성 대화 무료 `[검증됨: Gemini Live 무료]`
- **[예시] 프롬프트**:
  ```
  너는 친절한 영어 회화 선생님이야. 카페에서 주문하는 상황으로 롤플레이하자.
  내가 틀리면 대화를 끊지 말고, 대화 끝에 실수 3개만 골라 고쳐줘.
  내 수준은 왕초보. 쉬운 단어로.
  ```
- **체감 효과**: 사람 앞에서 못 하던 연습을 부담 없이 24시간.

#### ④ 습관·목표 코칭 (Habitree ReadTree와 시너지)
- **[예시] 프롬프트**:
  ```
  나는 '매일 20분 독서' 습관을 만들고 싶어. 3주 계획을 세워주고,
  작심삼일 안 되게 '실패했을 때 다시 시작하는 규칙'도 정해줘.
  매일 아침 나한테 물어볼 체크 질문 1개도.
  ```
- **체감 효과**: 자기계발 결심을 "실행 가능한 계획"으로. → **ReadTree 연계 콘텐츠로 자연스럽게 이어짐.**

### 3-4. 창작/이미지 🎨

#### ① 말로 설명 → 한글 글자 박힌 포스터·썸네일·카드뉴스 ⭐
- **도구**: Google Gemini 앱 이미지 생성(Nano Banana 2 / Nano Banana Pro). 무료 할당량 내 생성·편집, 소진 시 기본 모델로 폴백. `[검증됨]`
- **핵심 강점**: 이미지 안에 **여러 언어의 정확하고 읽히는 텍스트**를 렌더링(짧은 태그라인~긴 문단), 포스터/목업에 다양한 폰트·질감. `[검증됨]`
- **[예시] 프롬프트**:
  ```
  독서 모임 홍보 포스터를 만들어줘.
  - 큰 제목(한글): "함께 읽는 저녁"
  - 부제: "매주 목요일 저녁 8시, 온라인"
  - 분위기: 따뜻하고 아늑한, 책과 커피
  - 세로형(SNS 스토리 비율). 글자 또렷하게.
  ```
- **체감 효과**: 디자인 못 해도 5분 만에 "쓸 만한" 결과물 → 창작 영역 최고 아하 모먼트.
- ⚠️ 상업적 사용 시 각 도구의 이용약관/저작권 확인. 사람 얼굴·유명 캐릭터·로고 도용 주의.

#### ② 사진 편집(요소 추가/제거/스타일 변경)을 말로
- **[예시] 프롬프트**: `이 사진에서 배경의 전선을 지워주고, 하늘을 맑은 노을로 바꿔줘.`
- **체감 효과**: 포토샵 없이 대화로 편집.

#### ③ SNS/블로그 콘텐츠 세트 한 번에
- **[예시] 프롬프트**:
  ```
  '무료 AI로 시작하는 첫걸음' 주제로 인스타 카드뉴스 5장 기획안을 줘.
  각 장의 (한 줄 제목 / 핵심 문구 / 이미지 설명)과
  게시글 캡션·해시태그 10개까지.
  ```
- **체감 효과**: 기획→문구→이미지 설명까지 원스톱 → 1인 크리에이터/소상공인에 강력.

## 4. 타겟별 커리큘럼 (강의 편성안)

## 5. 콘텐츠 제작 원칙 (Habitree 톤앤매너 적용)

4. **"저도 그랬어요" 동료 시선** — 가르치는 사람 아닌, 먼저 해본 사람.
5. **안전을 부담이 아니라 배려로** — 겁주지 말고 "이것만 지키면 안심".
6. **날짜 명시** — "2026년 7월 기준, 무료 한도는 바뀔 수 있어요."
7. **ReadTree·LINKMAP 연결** — 학습/습관 → ReadTree, 만들기 → LINKMAP로 서비스 내 순환.

## 8. ⚠️ 모든 콘텐츠 공통 "안전·한계" 고지 (필수 삽입)

### 8-1. 개인정보 `[검증됨]`
- 무료 AI 대화는 **기본적으로 AI 학습 데이터로 수집·이용될 수 있음** (국내 개인정보보호위원회 2025 안내서 + OpenAI 도움말 확인).
- **대응**: ① 민감정보(주민번호·계좌·비밀번호·회사 기밀·타인 정보) 입력 금지 ② 설정에서 학습 이용 **옵트아웃**(예: ChatGPT 설정 > 데이터 컨트롤).

### 8-2. 환각(사실 오류) `[검증됨]`
- AI는 없는 사실을 진짜처럼 생성. 도메인별 평균 환각률 **의료 15.6% · 법률 18.7%**(전 모델 평균).
- 단, **최상위 모델(ChatGPT·Claude·Gemini)은 의료 4.3%·법률 6.4%로 약 3배 낮음** → "그래도 확인은 필수"로 균형 있게 안내.
- **대응**: 숫자·인용·링크·의료·법률·날짜는 원문/공식 출처로 재확인. "AI는 초안, 판단은 사람."

### 8-3. 저작권/상업적 사용
- AI 생성 이미지의 상업적 사용은 각 도구 약관 확인. 실존 인물·브랜드·캐릭터 무단 생성 주의.

## 9. 아직 확인이 필요한 부분 (콘텐츠화 전 보강)

이번 리서치에서 **검증 실패했거나 근거가 약한** 항목 — 콘텐츠에 단정적으로 쓰지 말 것:

1. **업무 도구(엑셀 자동화·PPT·회의 전사)별 정확한 무료 한도** — MS Copilot·Perplexity 관련 수치 주장들이 검증 실패. 실제 도구를 직접 테스트해 최신 한도 확인 후 기재.
2. **시니어 대상 공공/오프라인 무료 AI 교육 사례** — "카카오임팩트 우리동네 시니어 디지털 스쿨" 주장은 검증 실패(🚫 사용 금지). 실재하는 국내 시니어 AI 교육 사례는 별도 확인 필요.
3. **각 활용법의 실증적 '체감 효과' 수치**(완주율·만족도) — 벤더 마케팅 외 독립 데이터 부족. Habitree 자체 소규모 사용자 테스트로 근거 확보 권장.
4. **한국 계정에서의 기능 실제 가용성** — Deep Research, Nano Banana Pro 등 일부 기능의 한국 접근성·한국어 성능은 제작 시점에 직접 확인.
