환경변수 · 수동 설정 가이드

AI가 자동화할 수 없고 사용자가 직접 발급·설정해야 하는 값들. 각 항목의 무엇 · 어느 메뉴(경로) · 입력할 값 · 단계별 방법 · 현재 상태를 한곳에 담았습니다. 이 페이지만 보고도 모든 설정을 끝낼 수 있도록 구성했습니다.

🔒 시크릿/키의 실제 값은 여기에 저장되지 않습니다. 이 보드는 "어디서 받아 어디에 넣는지"만 안내합니다. 값은 .env.local(로컬, 커밋 금지)과 Cloudflare 빌드 변수에 직접 입력하세요. 체크 상태는 브라우저에만 저장됩니다. 각 항목의 "설정 방법 자세히 ▾"를 펼치면 단계별 안내가 나옵니다.

⚡ 지금 당장 해야 할 순서 (배포 살리기)

  1. Cloudflare 빌드 변수 4개 등록 — Supabase 키 등 (아래 Cloudflare 섹션 cf-vars). 안 하면 빌드 실패.
  2. 커밋·푸시로 첫 빌드 트리거 — npm run ship "fix: deploy" → 자동 빌드.
  3. Builds 탭에서 성공 확인 → *.workers.dev 접속.
  4. 이후 커스텀 도메인 연결, 필요 시 Stripe/Resend 등 나머지.
