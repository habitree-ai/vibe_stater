## 작성 규칙 (삭제하고 사용)
1. 캡처는 `service-setup/<service>/screens/`에 `01_*.png`부터 순번으로 저장.
2. 화면 자막(STEP n/N · 제목 · 설명)과 본 문서의 단계가 **1:1로 일치**해야 함.
3. 버튼/메뉴는 **영문 라벨 그대로 + 한글 설명** 병기 (예: **Onboard a domain** 클릭).
4. 가입·비밀번호·결제·2FA·권한 변경은 **사용자 직접 수행** 명시. 봇 차단 우회 금지.
5. 재현(reconstruction) 화면은 "재현(representative)"임을 반드시 표기.
6. 개인정보(실이메일·계정ID·실데이터) 노출 금지 → `example@email.com` 등으로 마스킹.
