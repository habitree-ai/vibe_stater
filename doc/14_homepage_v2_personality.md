# 14. 메인페이지 v2 고도화 기획 — "사람이 만든 페이지"

> 목표: 현재 "Editorial × Tech Premium" 랜딩(v1)을 **개성·디테일을 강화해 '템플릿/AI 생성물 티'를 제거**하는 방향으로 전면 고도화한다.
> 범위: 홈(`/`) 전 섹션 + 디자인 토큰/모션/카피.
> 산출물: 본 기획서 + 비주얼 시안(`doc/mockups/home-v2.html`).
> 근거: 2025–2026 트렌드·안티-AI 디자인 리서치(본 문서 §7 출처).

---

## 0. 한 줄 전략

> **"emerald 단색 + 보편 그라데이션"을 버리고, 따뜻한 숲톤 + 그레인 + 에디토리얼 타이포 + 1인칭 손글씨 디테일로 "최동혁이라는 사람이 직접 짠 페이지"를 만든다.**

리서치 핵심 통찰: AI 슬롭은 "학습 데이터의 중앙값"이다(보라 그라데이션·Inter·중앙정렬 hero·아이콘 3박스·균일한 라운드/그림자). **디폴트를 의도적으로 깨는 것**이 곧 '사람이 만든 티'다.

---

## 1. 현재(v1) 진단

v1은 이미 완성도가 높다. 오로라/그레인/마퀴/Reveal/벤토 등 트렌드 요소를 갖췄다. 그러나 **"잘 만든 템플릿"의 인상**에 머문다. 구체적 약점:

| # | 약점 | 진단 | 왜 'AI/템플릿 티'인가 |
|---|---|---|---|
| 1 | **중앙정렬 Hero + 듀얼 CTA + 칩** | 가장 흔한 AI 히어로 공식 | 리서치의 No.2 AI 신호("중앙 hero + 버튼 + 아래 그리드") |
| 2 | **emerald→teal→violet 그라데이션 텍스트** | `.text-gradient`가 violet까지 감 | 보라/인디고 그라데이션 = No.1 AI 신호 |
| 3 | **얼굴/실물의 부재** | About이 "인용문 + 타임라인 3"으로 추상적 | 개인 브랜드인데 '사람'이 안 보임 |
| 4 | **균일한 리듬** | 모든 섹션이 `max-w` 중앙 + 같은 라운드/그림자 | 수직 리듬 변주 없음 → 템플릿 느낌 |
| 5 | **카피의 일반성** | "함께 키워가는 도구들", "신뢰감" 톤 | 1인칭 구체성·일화·의견 부족 |
| 6 | **목업 비중 과다** | LINKMAP 브라우저 목업이 hero를 점령 | 정작 '실제 화면 데모'는 정적 |
| 7 | **손맛 요소 0** | 전부 기하학적 도형·그라데이션 | 손그림·주석·인간 지문 부재 |

> v1을 버리는 게 아니라, **위 7개를 외과적으로 교체**해 "정제된 비쌈"에서 "사람 냄새 나는 비쌈"으로 이동한다.

## 2. 디자인 방향 (4대 지렛대)

리서치가 지목한, 비용 대비 효과가 가장 큰 4가지에 집중한다.

### 지렛대 1 — 컬러: "네온 emerald" → "숲/이끼 + 따뜻한 크림 + 잉크"
- primary emerald를 **차가운 네온이 아닌 깊은 숲톤**으로 (oklch hue를 163→150~155, 채도 약간 낮춤).
- 배경: 순백 대신 **따뜻한 크림/페이퍼**(이미 `oklch(0.992 0.003 95)` — 더 따뜻하게 95→85 hue).
- 액센트: violet 제거. 대신 **번트 오렌지/테라코타**(따뜻한 보색)로 "손맛·아날로그" 신호. CTA·강조 밑줄·도장(stamp)에만 소량.
- **지배색 1 + 날카로운 액센트 1** 원칙(소심하게 분포된 팔레트 금지).

```
--paper:    oklch(0.985 0.008 85)   /* 따뜻한 크림 */
--ink:      oklch(0.20 0.015 150)   /* 잉크 그린블랙 (검정 대신) */
--forest:   oklch(0.52 0.11 152)    /* 깊은 숲 emerald (primary) */
--moss:     oklch(0.70 0.10 150)    /* 밝은 이끼 (보조) */
--clay:     oklch(0.64 0.14 47)     /* 테라코타 액센트 (손맛) */
--grain:    SVG fractalNoise, opacity 0.5, soft-light
```

### 지렛대 2 — 타이포: Inter/Space Grotesk 의존 탈피 → 에디토리얼 페어링
- **헤드라인: Fraunces(가변 세리프)** — softness/opsz/wonk 축으로 "quirky↔elegant" 조절. 본 브랜드의 '에디토리얼 + 핸드크래프트' 양면성에 최적. (현재 Instrument Serif 액센트를 Fraunces로 승격)
- **본문/UI: Pretendard 유지**(한글 본문) + 라틴은 **Geist** fallback. (Inter 회피)
- **극단 위계**: 헤드라인 weight 600~900 vs 캡션 11~13px. 스케일 점프 1.5배가 아니라 **3배 이상**.
- **letter-spacing 조이기**: 헤드라인 `-0.03em` (가장 저렴한 '타이포세팅한' 신호).
- old-style numerals·ligature `font-feature-settings` 활성.

### 지렛대 3 — 레이아웃: 균일 그리드 → 수직 리듬 변주 + 의도적 비대칭
- "커스텀 느낌의 80%는 수직 리듬 변주에서 나온다."
- 섹션 리듬 설계: **풀블리드 1 + 큰텍스트 짧은 섹션 1 + 비대칭 오프그리드 1**을 교차.
- Hero를 **좌측 정렬 비대칭**으로(중앙정렬 탈피). 우측은 실데모/사진이 1~2도 기울어 오버랩.
- 이미지/카드 미세 회전(`rotate(-1deg~1.5deg)`), 음수 마진 오버랩. 단 **focal point는 하나** 유지(통제된 카오스).
- 제품 라인업은 **벤토 그리드 유지**(이 패턴이 맞는 자리), **글/저널은 벤토 금지** → 에디토리얼 세로 흐름.

### 지렛대 4 — 인간 지문: 손그림 1 + 1인칭 카피 + 실물 사진
- **손그림 SVG 주석** 1세트: 핵심 단어 밑줄(wobble), 동그라미, 화살표. ("연결하는"에 손밑줄, 통계에 화살표)
- **본인 사진**(로고 아님): About에 폰으로 찍은 듯한 사진 + 그레인 15~20%. (자산 준비 전까지 placeholder 프레임 + "사진 들어갈 자리" 명시)
- **1인칭 구체 카피**: "신뢰감 있는" 같은 형용사 대신 일화·수치·의견. (§4 카피 가이드)
- **도장(stamp)/뱃지**: "교육용 100% 무료", "함께 만드는 중"을 고무도장 질감으로.

## 3. 섹션별 고도화 명세 (전체 페이지)

순서도 손본다. **신뢰 → 사람 → 증거(실서비스) → 가치(무료) → 행동** 흐름.

### ① 헤더
- 투명→스크롤 시 글래스 유지. 단 **워드마크에 손맛**(나무 아이콘을 1px 손그림 라인으로).
- nav 항목 hover 시 **테라코타 손밑줄** 슥 그어짐(200ms).

### ② Hero — 중앙정렬 탈피, 비대칭 에디토리얼
- 레이아웃: **좌 60%(텍스트) / 우 40%(실데모 카드, 1.5도 기울임)**.
- 헤드라인: Fraunces 초대형, "읽고 · 만들고 · **연결하는**" — "연결하는"에 **손그림 밑줄 + clay 색**.
- 위계 극단화: 헤드라인 옆 작은 11px 캡션("개인 프로젝트 · 2024–").
- CTA: primary 1개("무료 자료 받기")만 강조 + 텍스트링크 1개(보라 글로우 그림자 제거, 대신 solid forest + 미세 그레인).
- 배경: 오로라 → **종이 그레인 + 옅은 숲 글로우 1개**(보라 오브 제거).
- 신뢰지표를 **손글씨 숫자 느낌**으로: "지금 만드는 중 · 커피챗 37/100".

### ③ Marquee (키워드 흐름) — 유지, 톤만 조정
- violet 칩 제거, clay/forest 2색 교차. 구분자 ✦ → 손그림 별 또는 `·`.

### ④ About — "사람이 보이는" 섹션 (최우선 개편)
- **2단 비대칭**: 좌측 본인 사진(기울임 + 그레인 + 폴라로이드 캡션), 우측 1인칭 자기소개.
- 추상 인용문 → **구체적 자기소개 3~6문장**(무엇을/누구를 위해/구체 크레딧 1개: "제조 ERP PM, AX 담당").
- 타임라인 3개는 유지하되 **년도를 손글씨 톤**, 카드 미세 회전.
- 키워드 칩 → **고무도장 스타일**.

### ⑤ Services (실서비스) — 벤토 + 실데모
- LINKMAP/ReadTree/YouTube를 **벤토 그리드**(1 大 + 2 小).
- 큰 카드(LINKMAP)에 **10~15초 UI 루프 영상**(또는 자동재생 GIF placeholder). 정적 목업 대체.
- hover 시 **커스텀 커서 "열기 →"** morph, 카드 1도 기울기 보정(hover에 수평 복귀).
- 태그(#환경관리 등)는 손글씨 톤.

### ⑥ Resources (무료 자료/커피챗) — 가치 강조 + 도장
- "교육용 100% 무료"를 **고무도장 뱃지**로 시각화(clay 잉크 번짐).
- 커피챗 진행바(37/100)를 **손그림 게이지** 느낌으로, 숫자 강조.
- 카드 리듬 변주(3개 균일 → 1개 와이드 피처 + 2개).

### ⑦ Journal (최근 글) — 벤토 금지, 에디토리얼 세로
- 첫 글 **大 피처드**(세리프 제목 + 발췌) + 나머지 컴팩트 리스트. 시간순 명확.
- 날짜 old-style numerals, 카테고리 라벨 손밑줄.

### ⑧ Education CTA — 다크 잉크 밴드 유지, 강화
- 배경 잉크그린(`--ink`) + 그레인. 5단계 프로세스는 **손그림 번호(01~05)** + 점선 연결선.
- 카피 1인칭화: "코드를 몰라도, 제가 했던 그대로 따라오시면 됩니다."

### ⑨ Newsletter / 문의 — 정제
- 글래스 2분할 유지. 인풋에 손그림 밑줄 placeholder. 보라 글로우 제거.

### ⑩ Footer — 대형 워드마크
- 거대 "habitree" 워드마크(Fraunces) + 미세 그레인. 손그림 나무 1컷.

## 5. 모션 / 인터랙션

- **Staggered page load**: hero 요소 `animation-delay`로 순차 등장(한 번의 잘 짜인 로드 > 산만한 마이크로). 이미 `Reveal` 있음 — 재사용.
- **커스텀 커서**: 서비스 카드 hover 시 라벨 원("열기 →") morph(경량 JS, 200ms).
- **손밑줄 draw-on**: nav/키워드 hover 시 SVG path `stroke-dashoffset` 애니메이션.
- **미세 기울기 보정**: 카드 default 1도 기울임 → hover 시 0도(살아있는 느낌).
- **스크롤 내러티브(선택)**: "읽고 → 만들고 → 연결" 3단계 순차 공개.
- **접근성**: `prefers-reduced-motion` 전부 존중(이미 globals.css에 분기 존재 — 신규 모션도 포함).

## 6. 구현 메모 (현 아키텍처 기준)

- 홈은 `src/generated/homeMarkup.ts`(자동생성 문자열) + `HomeEnhancer`(client) 구조. 고도화 마크업도 동일 파이프라인으로 주입하거나, 섹션 컴포넌트(`src/components/sections/*`)로 회귀해 SSR화 택1.
- **토큰**: `globals.css` `:root`에 §2 신규 변수 추가, `.text-gradient`에서 violet stop 제거.
- **폰트**: `next/font/google`로 Fraunces 로드, `--font-serif`/`--font-display` 매핑 교체.
- **신규 유틸**: `.hand-underline`(SVG mask), `.stamp`(고무도장), `.tilt`/`.tilt:hover`, `.cursor-label`.
- **성능**: 그레인·글로우는 CSS 우선, 영상은 lazy + `prefers-reduced-motion` 시 포스터 정지.
- 검증: `npm run build` 통과, 모바일/데스크톱 반응형, reduced-motion 동작.

## 8. 자체 검토 의견 (디자인 비평 + 접근성)

시안을 시니어 디자이너/접근성 관점에서 비평하고, 핵심을 시안에 반영했다.

### 검증된 명도대비 (WCAG 2.1 AA, oklch→sRGB 실측)
| 조합 | 대비 | 판정 |
|---|---|---|
| forest / 크림 | 5.0:1 | 본문 통과 |
| ink-soft / 크림 | 8.7:1 | 통과 |
| **clay(구 `0.60 0.15 47`) / 크림** | **4.0:1** | 일반 텍스트 **불합격** |
| clay-soft / 크림 | 2.7:1 | 텍스트 전면 불합격 |
| moss / 다크 ink | 7.2~7.8:1 | 다크 위 통과 |
| moss / 흰 배경 | 2.4:1 | 밝은 배경 텍스트 **금지** |

### 시안에 반영 완료(✓)
- **clay 텍스트 대비**: `--clay`를 `0.50 0.13 47`로 어둡게 → 크림 위 4.5:1↑ 확보(kicker·태그·도장·저널 라벨 일괄 개선). clay-soft는 도형 전용으로 명시.
- **이모지 로고 제거**: 🌳(OS 기본 이모지 = AI/템플릿 티 1순위)를 **1px 손그림 나무 inline SVG**로 교체(헤더·푸터). `aria-label` 부여, 장식 SVG는 `aria-hidden`.
- **마퀴 ✦ 클리셰**: 손그림 별 SVG 마스크로 교체(기획 §3-③ 약속 이행).
- **키보드 포커스**: 전역 `:focus-visible` 링 추가, 헤더 워드마크를 `<a>`로.

### 실제 코드화 시 남은 보완(권장 우선순위)
1. **moss는 밝은 배경 텍스트로 절대 사용 금지** — 다크 밴드 위에서만. 흰/크림 위 강조는 forest/clay(어둡게)로.
2. **인터랙티브 시맨틱**: 서비스 카드를 `<div>` → `<a href>`로(키보드 접근·실행), 모바일 햄버거 메뉴 구현, 터치 타깃 44px 확보. 커스텀 커서는 장식으로만(라벨은 카드 내 텍스트로도 노출).
3. **수직 리듬 변주(기획 §2-3 핵심)를 실제로**: 현재 시안은 대부분 동일 `max-w + py-96`. About 풀블리드 확장 / 큰텍스트 짧은 전환 섹션 1개 삽입 / Education 풀블리드 다크로 리듬 차이를 만들 것.
4. **손맛을 1종(밑줄) → 다종으로(절제)**: 동그라미·화살표 주석 1~2개 추가, `.stamp`를 실제 잉크번짐 질감으로 업그레이드. 단 한 섹션(Resources)에 강조요소 과밀 → focal point 1개로 정리.
5. **위계 스케일**: 카드 h3(1.8rem)가 섹션 h2와 근접 → 섹션 제목/본문 점프를 3배 이상으로 더 벌릴 것.
6. **사진 placeholder**를 실제 `<img alt="…">` 구조로 설계(CSS content 텍스트는 접근성 트리 누락).

> 총평: 방향·기획은 탄탄. 시안은 "사람이 만든 티" 약 70% 달성 → 이모지·대비·포커스 수정으로 끌어올렸고, 남은 30%(수직 리듬·손맛 다양화·시맨틱)는 실제 코드화 단계의 작업으로 분리한다.
